Coin Hunt (Python)

About

"Coin Hunt" is a 2D game made with Python. This project was created as the final assignment of the advanced Python programming course at the Department of Computer Science at the University of Helsinki.

In the game, the player picks up as many coins as possible while dodging monsters. Survival time is tracked, and the player needs to stay alive as long as possible. They can buy temporary upgrades to aid them with the coins they have collected.

Introduction

This game was made as a final assignment of the course at the Department of Computer Science at the University of Helsinki.

There is an introductory course as well as an advanced Python programming course. These courses are open to everyone and provide a good foundation in programming. Attendees receive a certificate upon completion.

Technical Details

The game was developed in Visual Studio Code using Python with the Pygame extension. All course students were provided with the same four images, and we came up with the game ideas ourselves.

Outcome and Impact

The games were distributed to other students for peer reviews, and I received positive feedback. One review mentioned that the game was 'creative' and had 'a good tempo'. Another comment noted that the code was well-organized. The overall score was 4.67/5.

What I Learned

I learned a lot about good programming practices and approaches in this course. The course covered basic data types, methods, loops, files, recursion, object-oriented programming, and much more.
