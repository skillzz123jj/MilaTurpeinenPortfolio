Värityskirja (C#, Unity)

About

"Värityskirja" (translated as "Coloring book") is a 2D game developed using Unity. This project was created as part of an internship at the Finnish Association on Intellectual and Developmental Disabilities (FAIDD).

In the game, players choose a picture from a collection of simpler and more detailed images to color in. Users can then download their image once it's finished.

Project Info

Internship project 5 Months 5 Lead Programmer Papunet's website

Introduction

This coloring game was made as an internship project with Finnish Association on Intellectual and Developmental Disabilities (FAIDD) as apart of my studies at Taitotalo. I helped one of the teams get their project started and eventually became the lead programmer.

Our teams were one the first ones to be apart of FAIDD's mission to remake their old games so this game will be used as a model for future games for their revamped website Papunet.

Technical Details

I was responsible for the gameplay and menu programming using C# and Unity. I also did version control for our team in git and shared and updated builds of the game to the customer using netlify.

Since the game was published on a website it was important that the game was optimized for mobile and desktop performance and scaling wise.

Collaboration and Accessibility

Games being accsessible for everyone is important and this collaboration with Papunet allowed me to learn a lot about it.

We were in close contanct with Papunets accessibility experts to make sure all aspects of the game were up to the player's needs.

Since the game is all about accessibility and learning I implemented the ability play with either mouse, keyboard or mobile.

Challenges and Solutions

The most challenging aspect of this game was implementing the option to download the images once they were ready. This required a lot of research and testing, and ultimately, I used a JavaScript function to initiate the download in this script.

Outcome and Impact

The game was well-received by FAIDD and contributed to the their mission of updating the websites games.

Used as a model for updating other games in the FAIDD's collection.

What I Learned

This project was one of my first significant ventures into game development, and it provided me with invaluable learning experiences. I gained foundational skills in programming, game design, graphical systems, and Unity, as well as experience with version control and collaboration.

One of the most important lessons I learned was the significance of accessibility in games and media. Working closely with knowledgeable professionals and receiving active feedback helped me understand the critical role accessibility plays in creating inclusive and engaging experiences.

Overall, this project was a pivotal moment in my career, broadening my understanding of game development and underscoring the importance of continuous learning and adaptation.
